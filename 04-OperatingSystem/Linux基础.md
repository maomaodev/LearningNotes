## SRPM 的使用：rpmbuild

rpmbuild 命令被用于创建 rpm 的二进制软件包和源码软件包。

| 目录                   | 含义                                                      |
| ---------------------- | --------------------------------------------------------- |
| /root/rpmbuild/SPECS   | 放置软件的配置文件                                        |
| /root/rpmbuild/SOURCES | 放置软件的原始文件（*.tar.gz文件）及config配置文件        |
| /root/rpmbuild/BUILD   | 放置编译过程中的某些缓存数据                              |
| /root/rpmbuild/RPMS    | 放置成功编译后打包完成的文件，包含x86_64、noarch 等子目录 |
| /root/rpmbuild/SRPMS   | 以 SRPM 方式发布时，放置 SRPM 封装的文件                  |



| 参数          | 含义                                                         |
| ------------- | ------------------------------------------------------------ |
| Summary       | 本软件的主要说明                                             |
| Name          | 本软件的软件名称                                             |
| Version       | 本软件的版本                                                 |
| Release       | 该版本打包的次数说明                                         |
| License       | 本软件的授权模式                                             |
| Group         | 本软件安装时，主要放置于哪个软件群组中（yum grouplist）      |
| URL           | 本源代码的官方网站                                           |
| SourceN       | 本软件的来源，若有多个软件来源，会以 Source0、Source1 来处理源代码 |
| PatchN        | 作为布丁的 patch file，可以有多个                            |
| BuildRoot     | 设置作为编译时，该使用哪个目录来缓存中间文件                 |
| Requires      | 可选，                                                       |
| BuildRequires | 可选，编译过程中所需要的软件                                 |



| 参数            | 含义                            |
| --------------- | ------------------------------- |
| %description    | 软件的简短说明                  |
| %define         | 定义 spec 全局变量              |
| %prep           | 预处理脚本，rpm 安装前执行操作  |
| %build          | 构建                            |
| %install        | 安装                            |
| %post           | rpm 安装后执行的脚本            |
| %preun、%postun | 卸载 rpm 包前后要做的事         |
| %files          | 定义哪些文件或目录会放入 rpm 中 |
